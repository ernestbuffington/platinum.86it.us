Setup Platinum Nuke Pro

Setting up your Platinum Nuke Pro Website can be done 2 differant ways. You can manually install it or run the installer. It all depends on your preferances.

I will first cover how to install the Platinum Nuke Pro CMS manually first. Please use the followng checklist to ensure you are ready to begin.

  • You have a Database created and a username and password as well.
  • Ensure you have filled out your root/config.php with the correct credentials (If you need help editing your config.php refer to the Preperation Docs).
  • Make sure all files have been uploaded to your root folder of your Webhost and permissions set. See permissions settings below.

Permissions Settings

  • Set permissions on folder /files to 777.
  • Set permissions on folder /files/thumbs to 777.
  • Set permissions on folder /modules/coppermine/albums to 777.
  • Set permissions on file /includes/smtp.php to 666.
  • Set permissions on file /config.php to 644.
  • Set permissions on folder /modules/NukeC30/imageads to 777.
  • Set permissions on folder /modules/NukeC30/imagecatg to 777.
  • Set permissions on folder /modules/Forums/cache to 777.
  • Set permissions on folder /modules/Forums/images/avatars to 777.
  • Set permissions on file /modules/Forums/language/lang_english/lang_faq.php to 666.
  • Set permissions on file /modules/Forums/language/lang_english/lang_rules.php to 666.
  • Set permissions on folder /modules/Supporters/images/supporters to 777.
  • Set permissions on folder /modules/Downloads/dl to 777.
  • Set permissions on folder /modules/Member_Application/admin/update.log to 666.
  • Set permissions on folder /modules/PrivateDownloads/loads to 777.
  • Set permissions on folder /modules/HTML_Newsletter/archive to 777.
  • Set permissions on folder /images/CZUser to 777.
  • Set permissions on folder /images/CZUser/admin to 777.
  • Set permissions on folder /modules/Link_Us/buttons to 777.
  • Set permissions on folder /modules/Universal/images/uploaded to 777.
  • Set permissions on folder /userfiles to 777.
  • Set permissions on folder /userfiles/files to 777.
  • Set permissions on folder /userfiles/flash to 777.
  • Set permissions on folder /userfiles/images to 777.
  • Set permissions on folder /includes/snavi to 777.
  • Set permissions on file /modules/Forums/glance_config.php to 777.

Once you have the above checklist correct you now need to go to your Cpanel provided by your Webhost and then to your PHPmyAdmin and select the Database you created for your Platinum Nuke Pro install and then click on the import option in the top menu. (See Example)

Once the screen shows your import options you simply hit the "choose file" button and go to your folder on your computer where you first found and edited your config.php and select your SQL file. Just look for the .sql extension and start your import of the file.

Once you get a message that the import was successfull you can visit your website.

You should now be able to see your website. If you can view your website your next step is to setup your administrator account. To do this simply point your browser to (http://www.yoursite.com/admin.php). Replace "yoursite" with your sites actual name. Fill in the information needed in each box making note of the username and password. You will need those to login to your admin account and user account.

The second option to install Platinum Nuke Pro is to run the Platinum Nuke Pro Installer. See steps below to install using the Installer.

  • Create your MySQL Database and assign a user and password to it.
  • Please write down the Database name, username and password to use in the installer.
  • Upload all files to your webhost.
  • Then you simply navigate to your site and follow the on screen instructions.
  • The installer will set your permissions.
  • The installer will create your config.php.
  • The installer will also create your God admin account and matching user account if selected.
  • The installer will set your sites name, URL and sets your admin E-mail as well.

Now that we have the CMS installed you will notice on the main page in the top middle block it asks you to complete a couple more tasks. One is the set permissions, please do this just to allow it to check that i did not miss a folder in the docs here. Once that is done it asks you to install the IP to Country tables, please do that as well. Lastly it asks you to install the tables for the Members Application Module tables. If you are going to use that feature then do so. If not then it will not affect your site to leave it be.
