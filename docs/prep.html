Preparing to install

In order to setup Platinum Nuke Pro the folowing prerequisites are necessary.

  • A Linux or Windows Box installed and working properly.
  • Apache Web Server (http://www.apache.org).
  • PHP version 5.3.x or better (mod_php) Apache module (http://www.php.net).
  • MySQL database server (http://www.mysql.com).

The following tasks will need to be completed before you attempt to run your new site.

  • Log into your website's control panel. (If you don't know where it's located, please contact your web host).
  • Create a new SQL database. (For example: name it nuke).
  • After creating your database please assign a username and password to that database and give that user full rights, (for example: username: plat, password: pass).
  • Edit config.php found in the Html folder (Upload Files From This Folder First) to match the settings of your SQL server and database..

Editing your config.php, this is where the above database name, username and password are needed. (Example Config) See below for descriptions of each field. Please note that if you are going to use the installer all you need to do is set up the database and the installer will do the rest.

  • $dbhost = "localhost"; (99% of the time this can be left as is).
  • $dbuname = "root"; (This is the username you created and assigned your database. Replace the letters inside the quotations with your username which for example we used plat above).
  • $dbpass = ""; (This is the password you assigned your above user which for the example was pass. Add that inside the qoutation marks).
  • $dbname = "dbname"; (This is where you will add the database name that you created. The example name was nuke. Again replace what is inside the quotation marks with your database name).
  • $prefix = "nuke"; (Does not need to be changed but can be useful if you want to run more then one Platinum Nukes on the same database).
  • $user_prefix = "nuke"; ( This does not need to be changed).
  • $dbtype = "MySQL"; (As Platinum Nuke only supports MySQL this can and should be left alone).
  • $sitekey = "7Ed+UratU+e9-eSwASW8s82MagUV*2"; (Be sure to change what ever is inside these quotes, leaving it as default is a security risk).

Now that you have all needed fields accurately filled in you can save your config.php. Upload ALL the files from the Html (Upload Files From This Folder First) to your website's doc root folder. Please contact your web host if you are not sure where your doc folder is located.
