NukeBlog Credits and Change Log

NukeBlog is the next generation of Journal for PHP-Nuke. NukeBlog is always in a state of development. Please visit Trevor.Net for the latest support, bug fixes, patches, and upgrades. NukeBlog is free for all to use! It is intended to build communities and friendships. Enjoy!
